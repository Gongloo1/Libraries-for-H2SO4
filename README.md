# H2SO4 Central Module Registry

Welcome to the official repository for **H2SO4** programming language extensions, standard library add-ons, and functional toolkits. 

Instead of manual cloning, these packages are designed to be scanned, tracked, and downloaded effortlessly using the custom **ION** package manager tool.

---

## Available Packages

| Package Name | Description | Includes |
| :--- | :--- | :--- |
| **`Advanced`** | High-performance functional programming paradigms and performance tracking. | `map`, `filter`, `reduce`, `range`, `benchmark` |

*More official packages and community modules will be registered here over time.*

---

## How to Install Packages

The cleanest way to fetch any package hosted in this repository is by running the **ION CLI client** tool on your local machine.

### 1. Run ION
Navigate to your local ION tool directory and run the engine via Deno:
```bash
deno run -A ION.ts
